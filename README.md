# Python2017学习手册

v 0.1

Python学习手册其实缘起2013年公司内部新手入职需要一本快速的上手资源，其实网络中关于Python学习的资源挺多的，但是适合java程序员快速转型思维模式的图书不多，这个小册子就是一本从java程序员思考的角度学习与想问题的快速入门手册，不求多深，只求快速。入门了你们再找大牛的资料，深入学习吧。希望对你们有帮助。
